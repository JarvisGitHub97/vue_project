# 基于Vue.js的小商城
---

## Introduction
项目基于Vue 经Webpack打包实现商城的发表评论，购买商品等基本功能，部署在本机的服务器上

---
## Webpack
- 基本配置 
- plugin配置：`webpack-dev-server`, `html-webpack-plugin`等
- loader配置：babel, 样式相关的loader等

## 数据
- 取：本地mock json格式文件
- 存：本地localStorage

## 引用
- 组件：Mint-UI组件库，boostrap
- 路由：vue-router
- 数据请求： vue-resource
- 数据共享：vuex

## Start
- 装包：`npm install`
- 启动：`npm run dev`
